A classic Tic-Tac-Toe game built using HTML, CSS, and JavaScript. This interactive web application allows two players to play the traditional game in a clean and responsive user interface.

Features
Interactive 3×3 Tic-Tac-Toe board
Two-player gameplay (Player X and Player O)
Automatic winner detection
Draw game detection
Reset/New Game functionality
Responsive design for different screen sizes
Simple and user-friendly interface

 Technologies Used
HTML5 – Structure of the game
CSS3 – Styling and responsive design
JavaScript (ES6) – Game logic and interactivity
